# AutoinsuranceQuote

This is an example of a form to make the auto insurance quote. The data entered by the user in this example are: Car brand, year and the type of insurance you want to quote.
You can review the operations that were taken to perform the calculations

## Starting 🚀

You just need to clone the project

### Pre requirements 📋

_This project uses the package manager npm and was bootstrapped with [Create React App](https://github.com/facebook/create-react-app)._  
**So you should have nodejs, npm and npx installed to be able to run the project**

### Installation 🔧

_To install project dependencies:_

```bash
npm install
```

## Run ⚙️

```bash
npm run start
```

## Built with 🛠️

- **[ReactJS]** - Framework
- **[NPM]** - Package manager
- **[StyledComponents]** - Used to generate styles
- **[ReactTransitionGroup]** - Make some animation
